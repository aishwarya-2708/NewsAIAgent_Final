# News AI Agent

An AI-powered news analysis and recommendation system.

This project processes news data and uses a local LLM to generate intelligent insights and responses.

--------------------------------------------------

REQUIREMENTS

- Python version 3.11 (recommended)
- Install all dependencies from requirements.txt
- Local LLM model: Llama 3.2:1b

--------------------------------------------------

SETUP INSTRUCTIONS

1. Install Python 3.11

Make sure Python version 3.11 is installed on your system.

2. Install Required Libraries

Install all the dependencies using the following command:

pip install -r requirements.txt

3. Start the LLM Model

Before running the application, start the Llama 3.2:1b model.

Example (if using Ollama):

ollama run llama3.2:1b

4. Run the Application

After the model is running, start the project using:

python app.py
